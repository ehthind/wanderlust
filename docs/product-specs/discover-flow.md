# Discover Flow

The first product slice centers on a destination-first Discover experience that feels editorial, immersive, and practical enough to plan against immediately on iPhone.

The iPhone Discover surface is a vertically paged, full-screen feed with three curated cards:
- Paris
- Kyoto
- Mexico City

Each card is image-led, supports lightweight local saving, and hands directly into native trip creation plus the stay-availability workspace. Discover remains the first tab and the first surface users see when they open the app.

Swiping left from the currently visible feed card opens a full-screen destination profile for that same city. The profile uses the same saved state and planning action, restores the factual detail rows that were removed from the feed, and shows a static editorial grid of eight story tiles. Swiping right returns to the same feed card, and the global tab chrome is hidden while the profile surface is visible.
